# CryptoSafe Manager

Менеджер паролей с шифрованием.

## Описание проекта
CryptoSafe Manager — приложение для хранения паролей с SQLite базой данных. Цель проекта — создать безопасный, модульный и расширяемый менеджер паролей с последующей поддержкой аудита, импорта/экспорта и автоблокировки.

## Спринт 1: Фундамент
**Цель:** Создать безопасный, расширяемый фундамент с зашифрованной базой данных, модульной архитектурой и базовой оболочкой GUI.

### Реализовано в Спринте 1:

- Модульная архитектура (MVC): `core/`, `gui/`, `database/`, `tests/`
- Менеджер конфигурации (`config.py`) для путей и настроек
- SQLite база данных с 4 таблицами (`vault_entries`, `audit_log`, `settings`, `key_store`)
- Индексы для таблиц (поиск по названию, датам)
- Заглушка шифрования (XOR) с абстрактным классом (`EncryptionService`)
- Менеджер ключей (`KeyManager`) с методами derive/store/load
- Система событий (`EventBus`) для слабосвязанных компонентов
- Заглушка журнала аудита (подписка на события)
- Менеджер состояния (`StateManager`) для отслеживания сессии, буфера, таймеров
- Tkinter GUI с меню, таблицей и строкой состояния
- Переиспользуемые виджеты (`PasswordEntry`, `SecureTable`)
- Мастер первоначальной настройки (мастер-пароль, расположение БД, настройки шифрования)

## Установка
```bash
# 1.клонировать репозиторий
git clone https://github.com/Ray-altq/cryptosafe-manager.git
cd cryptosafe-manager

# 2.создать виртуальное окружение
python -m venv venv

# 3.активировать
# Windows
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4.установить зависимости
pip install -r requirements.txt

# 5.запустить
python run.py
```
